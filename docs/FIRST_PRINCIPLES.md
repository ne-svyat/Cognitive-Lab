Cognitive Lab — First Principles

Mission

The goal of this project is not to create another language model.

The goal is to research alternative cognitive architectures that are:

- understandable;
- modular;
- observable;
- measurable;
- replaceable;
- efficient.

---

Core Questions

1. Why should intelligence be monolithic?

2. Can memory exist outside parameters?

3. Can reasoning be decomposed into independent local worlds?

4. Can routing replace brute-force computation?

5. Can concepts become first-class objects instead of tokens?

6. Can every operation be visualized?

---

Research Rules

Every idea must answer:

- What problem does it solve?
- Why is the current approach insufficient?
- How can it be measured?
- What are the disadvantages?

---

Project Philosophy

Everything should be visible.

Everything should be replaceable.

Everything should be testable.

Nothing is sacred.
